# File Formats

## TetGen specific formats

<table width="80%" border="1" align="center">
<tr>
  <td><a href="../fformats_node">.node</a></td>
  <td>input/output</td>
  <td>a list of nodes</td>
</tr>
<tr>
  <td><a href="../fformats_poly">.poly</a></td>
  <td>input</td>
  <td>a <a href="../plc">piecewise linear complex</a></td>
</tr>
<tr>
  <td><a href="../fformats_smesh">.smesh</a></td>
  <td>input/output</td>
  <td>a simple <a href="../plc">piecewise linear complex</a> </td>
</tr>
<tr>
  <td><a href="../fformats_ele">.ele</a></td>
  <td>input/output</td>
  <td>a list of tetrahedra</td>
</tr>
<tr>
  <td><a href="../fformats_face">.face</a></td>
  <td>input/output</td>
  <td>a list of triangular faces</td>
</tr>
<tr>
  <td><a href="../fformats_edge">.edge</a></td>
  <td>output</td>
  <td>a list of boundary edges</td>
</tr>
<tr>
  <td><a href="../fformats_vol">.vol</a></td>
  <td>input</td>
  <td>a list of maximum volumes</td>
</tr>
<tr>
  <td><a href="../fformats_var">.var</a></td>
  <td>input</td>
  <td>a list of variant constraints for facets/segments</td>
</tr>
<tr>
  <td><a href="../fformats_neigh">.neigh</a></td>
  <td>output</td>
  <td>a list of neighbors</td>
</tr>
</table>
