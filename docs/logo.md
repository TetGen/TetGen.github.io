# The TetGen Logo

Intuitivly, this picture shows a three-dimensional simplex (a
tetrahedron) with a subdivision of the tetrahedron into four
disadjointed parts by selecting a point <em>C</em> inside it. <p>

It is also a Delaunay tetrahedralization of a set of four points in
three dimensions, which is trivial, and a Voronoi diagram of the same
point set when <em>C</em> is the circumcenter of this tetrahedron.<p>

More specifically, this Voronoi diagram consists of four Voronoi cells
corresponding to the four points, which are all unbounded. It has six
facets (shown in different colors), four edges (connecting at
<em>C</em>) and one vertex (<em>C</em>).<p>

Following are two more examples show the Voronoi diagrams
corresponding to the four points when one of the four points is
transformed.<p>
